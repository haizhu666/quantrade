# 上证50ETF期权回测系统

一个简化的 **上证50ETF（510050.SH）** 数据获取与期权回测框架。支持从 Tushare 拉取真实行情，也支持在无 API 权限时自动生成模拟数据进行本地测试。

---

## 功能特性

- **数据获取**：自动拉取 50ETF 日线及期权链行情，支持本地 CSV 缓存。
- **趋势策略**：基于价格与均线的金叉/死叉生成交易信号。
- **向量化回测**：简化版向量化回测引擎，按日遍历计算净值与盈亏。
- **降级兼容**：无 Tushare 付费权限时，自动 fallback 到模拟数据，保证项目可运行。

## 快速开始

### 1. 配置 Tushare Token（可选）

编辑 `config/settings.py`，将你的 Tushare Pro Token 填入：

```python
TUSHARE_TOKEN = "你的token"
```

> 如果你没有 Tushare 账号或没有对应接口的付费权限，程序会自动使用**模拟数据**运行，不影响功能体验。

### 2. 运行回测

```bash
python main.py
```

控制台将输出：
- 数据下载/缓存/模拟状态
- 每笔交易（买入/平仓）详情
- 回测结果（初始资金、最终资金、总收益率）
- 结果 CSV：`backtest_result.csv`

## 数据获取说明

### 真实数据（需 Tushare 权限）

项目会依次尝试以下接口：
1. `pro.fund_daily()` — ETF 基金日线
2. `pro.daily()` — 通用日线（fallback）
3. `pro.opt_basic()` + `pro.opt_daily()` — 期权合约及行情

> ⚠️ **注意**：Tushare 的 ETF/期权接口通常需要一定积分或付费权限。如果遇到 `"抱歉，您没有接口访问权限"`，程序会自动降级。

### 模拟数据（无权限时自动启用）

- **ETF 数据**：基于随机游走生成带趋势的收盘价序列，并构造 OHLCV 字段。
- **期权数据**：按平值附近 5 档行权价生成认购/认沽合约，每天价格加入随机扰动，模拟真实波动。

模拟数据仅用于**本地调试和框架验证**，不代表真实市场走势。

## 策略说明

### TrendFollowStrategy（均线趋势跟踪）

位于 `strategies/trend_follow.py`

**逻辑**：
- 计算 N 日移动平均线（默认 5 日）。
- **买入信号**：昨日收盘价低于均线，今日收盘价高于均线（金叉）→ 买入近月平值认购期权。
- **卖出信号**：昨日收盘价高于均线，今日收盘价低于均线（死叉）→ 平仓。

**参数**：
- `ma_window`：均线窗口，默认 `5`

## 回测说明

位于 `backtest/vectorized.py`

### 核心假设（简化版）

- 每日**收盘**按收盘价成交，暂不考虑滑点与手续费。
- 期权买入时支付权利金，卖出时收回权利金，盈亏 = `(卖出价 - 买入价) × 10000`（1张 = 10000份）。
- 持仓期间按当日期权市场价计算 NAV；若当日无该合约行情，则按成本价估算。

### 输出字段

`backtest_result.csv` 包含：

| 字段 | 说明 |
|------|------|
| `date` | 日期 |
| `nav` | 当日净值（现金 + 持仓市值） |
| `cash` | 当日现金余额 |
| `holding` | 是否持仓（1=有，0=无） |

## 注意事项

1. **模拟数据局限性**：当前生成的模拟 ETF 和期权数据仅供跑通流程， Greeks、隐含波动率、到期日等关键期权要素未建模。
2. **回测简化**：未考虑手续费、滑点、涨跌停、流动性、合约到期换月等现实因素。
3. **权限升级**：如需真实回测，请前往 [Tushare Pro](https://tushare.pro) 注册并积累积分/购买权限。
4. **投资有风险**：本仓库仅供学习研究，不构成任何投资建议。
